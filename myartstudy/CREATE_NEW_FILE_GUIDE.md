# GitHub の Create new file で作る手順

このフォルダ内の4ファイルを、GitHubリポジトリのルート直下に作成してください。

## 作成するファイル

1. `index.html`
2. `study-note-manifest.webmanifest`
3. `study-note-sw.js`
4. `study-note-icon.svg`

## コピペ手順

GitHubのリポジトリ画面で、各ファイルごとに次を行います。

1. `Add file` -> `Create new file`
2. ファイル名に上のファイル名を入力
3. `H:\download\myartstudy` にある同名ファイルを開く
4. 中身をすべてコピーしてGitHubの入力欄に貼り付け
5. `Commit changes`

## 注意

- 4ファイルは同じ階層に置いてください。
- `index.html` がトップページになります。
- GitHub Pagesを有効化すると、公開URLで開けます。
- Googleログインを使う場合は、Google Cloudの承認済みJavaScript生成元にGitHub Pagesの公開URLを追加してください。
